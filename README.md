# Инструкция по запуску проекта

## Требования

- PHP 7+
- MySQL
- XAMPP (Apache + MySQL)

---

## 1. Настройка подключения к базе данных

Проект использует **MySQL**.

Файл конфигурации подключения находится по пути:

Перед запуском необходимо указать корректные параметры подключения:

```ini
[database]
host = localhost
dbname = sql_console
user = root
password = secret
charset = utf8mb4
```

---

## 2. Запуск миграций

Для создания таблиц базы данных выполните команду из корня проекта:

```sh
php migrations/migrate.php
```

---

## Запуск проекта через XAMPP

1) Запустите Apache и MySQL в панели управления XAMPP.
2) Поместите папку проекта в директорию htdocs.
3) Откройте браузер и перейдите по адресу:

```
http://localhost/sql-console/public
```
